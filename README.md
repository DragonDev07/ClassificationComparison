## Installation & Usage

1. Clone the repository

```bash
git clone https://github.com/DragonDev07/ClassificationComparison.git
```

2. Install the required packages

```bash
pip install -r requirements.txt
```

3. Run main.py

```bash
cd src
python main.py
```
